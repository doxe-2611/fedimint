pub struct GatewayLdkClient {
    node: Arc<Node>,
    esplora_client: AsyncClient,
    event_handler_task_handle: JoinHandle<()>,
    htlc_stream_receiver_or: Option<Receiver<Result<InterceptHtlcRequest, Status>>>,
}

Fields§

§node: Arc<Node>

The underlying lightning node.

§esplora_client: AsyncClient

The client for querying data about the blockchain.

§event_handler_task_handle: JoinHandle<()>

A handle to the task that processes incoming events from the lightning node. Responsible for sending incoming HTLCs to the caller of route_htlcs. TODO: This should be a shutdown sender instead, and we can discard the handle.

§htlc_stream_receiver_or: Option<Receiver<Result<InterceptHtlcRequest, Status>>>

The HTLC stream, until it is taken by calling ILnRpcClient::route_htlcs.
